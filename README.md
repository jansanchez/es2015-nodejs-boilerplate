# es2015-nodejs-boilerplate
es2015 nodejs boilerplate. Includes: babel, xo, webpack and pm2

### Clone
```
git clone https://github.com/jansanchez/es2015-nodejs-boilerplate.git
```

### Install

```
npm install
```

### Npm Scripts (tasks)

Linter

```
npm run lint
```

Development

```
npm run dev
```

Production

```
npm run production
```
